  3. The full source and MCP server are at github.com/eusef/phils-personal-mcp

MCP Server (Optional)

For deeper integration, a stdio-based MCP server is available in the repo. Clone and build:

git clone https://github.com/eusef/phils-personal-mcp.git
cd phils-personal-mcp/server
npm install && npm run build
node dist/index.js
